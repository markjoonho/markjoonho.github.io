# Defining the Language of Propositional Calculus

- Set of Symbols (Characters): letters

  - Logical Symbols: $$(, ), \land, \lor, \implies, \neg$$
  - Variables: $$P_1, P_2,...,P_n$$

- Set of Expressions: finite sequences of symbols

  - $$p\lor q$$, (((\land \land)))

- Set of Propositions: words

---

$$X$$ = the set of expressions

$$A$$ = $$\{P_1, P_2, ..., P_n, ...\}$$.

$$
P $$= $$\{\frac{\alpha, \beta}{(\alpha \lor \beta)}, \frac{\alpha, \beta}{(\alpha \land \beta)}, \frac{\alpha, \beta}{(\alpha \implies \beta)}, \frac{\alpha}{(\neg \alpha)}\}
$$.

Words = Propositions = $$I(A,P)
$$

---

_**Examples of Propositions**_

$$P_1$$, $$P_{17}$$, $$(\neg P_{18})$$, $$(P_1 \lor P_{100})$$, $$(\neg (\neg P_{20}))$$, ...

---

A Construction sequence for an expresion $$\alpha$$ is a finite sequence $$\alpha$$ = $$\alpha_1$$, $$\alpha_2$$, ..., $$\alpha_n$$ such that each $$\alpha_i$$ is either a member of the core set A or it is the result of applying some $$f \in P$$ to $$\alpha_j$$, $$\alpha_k$$ where $$j$$, $$k < i$$ .

> Claim: For every $$X$$, $$A$$, $$P$$, and expression $$\alpha$$, $$\alpha \in I(A,P)$$ if and only if there exists a construction sequence for $$\alpha$$ .

---

_**Property 1**_

In every proposition, the number of left brackets '(' equals the number of right brackets ')'.

How can we show that some property T is satisfied by all members of I(A, P)?

(i) Every member of A have property T,

(ii) If $$\alpha_1, \alpha_2$$ has property T and $$f \in P$$, then $$f(\alpha_1, \alpha_2)$$ has T

If (i) and (ii) meets, then $$I(A,P) \subseteq T $$.
