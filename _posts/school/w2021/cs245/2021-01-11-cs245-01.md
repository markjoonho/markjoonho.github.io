## What is Logic?

It focuses on how to reliably get from assumptions to conclusions.

Example: $$F=ma$$

- Physician: True or False
- Mathematician: If $$F=ma$$ then $$a=\frac{F}{m}$$
- Logician: If whenever $$X=yz$$ then $$y =\frac{X}{z}$$. This statement works then if $$F=ma$$ then $$a=\frac{F}{m}$$

## Inductive Definitions of Sets

It is consists of followings:

1. Universe set: $$X$$
2. Core set: $$A\subseteq X$$
3. Set of Operations($$P$$): Functions from $$X\times X \times ...\times X \to X$$. Requires that $$P$$ is finite.

$$B\subseteq X$$ is closed under the operations in $$P$$, if for every $$b_1..bn \in B$$ and $$f \in P$$, $$f(b_1, ...,b_k) \in X$$

### $$I(A,P)$$

The set defined inductively from $$X,A,P$$ is denoted $$I(A,P)$$ is the smallest subset of $$X$$ that contains $$A$$ and is closed under $$P$$.

_**Exampel**_

$$X = Real\space Number , A= {5}, P = \{f1: x->x+1, f2: x->x-1\}$$

$$I(A,P) = All \space Integers$$

## Does such smallest set always exist? Yes.

$$I(A,P)= \cap\{B \subseteq X: A \subseteq B, B closed under P\}$$
