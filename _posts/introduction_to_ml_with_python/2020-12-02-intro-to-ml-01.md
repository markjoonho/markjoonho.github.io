# 1. Supervised Learning

There exists input and output sample data(training data). We predict the output from the input. Create model using the training data.

# 2. Type of Supervised Learning

## Classification

- Predict the class label.
- ex) binary classification(yes/no).
- ex) multiples classification(tiger/lion/cat/dog...).

## Regression

- Predict the real number.
- ex) predict the yearly income based on one's education level, age, or residence.
